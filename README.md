# Pegmus
Pegmus é uma extensao para salvar playlist do youtube

## Sobre

- Pegmus foi criado para ser uma forma simples e offline de salvar playlists
- Pegmus está limitado para funcionar somente em https://www.youtube.com/playlist?list=

## Melhorias pendentes

- ⏳ Salvar nome da playlist
- ⏳ Adicionar todas as musicas (atualmente salva somente carregadas)
- ⏳ Não salvar recomendações  da playlist
- ⏳ Adicionar suporte para urls https://www.youtube.com/watch?.....&list=....
